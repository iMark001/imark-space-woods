# Project Memory: Woods (iMark.space)

## Текущий стек и архитектура
- **Тип проекта**: Статический веб-сайт (Landing / B2B портал).
- **Технологии**: Vanilla HTML, CSS, JavaScript. Без фреймворков.
- **Хостинг/Деплой**: Netlify (через GitHub).

## Ключевые требования и задачи
1. **Мобильная адаптация**:
   - Жесткая фиксация `overflow-x: hidden` во избежание "прыгающего масштаба" на мобильных устройствах.
   - Корректное отображение таблиц: горизонтальный скролл с `position: sticky` для первых столбцов.
2. **Дизайн**:
   - Переход от мрачного стиля (глубокий черный/янтарный) к более лаконичному, премиальному и светлому дизайну (вдохновлено современными концептами из Figma).
   - Использование легких границ, воздушного пространства, микроанимаций (glassmorphism/софт-минимализм).
3. **Безопасность**:
   - Добавлен файл `_headers` для настройки базовой HTTP-защиты в Netlify (CSP, X-Frame-Options и др.).

## Лог изменений
- **Итерация 1 (Текущая)**: Настройка памяти проекта, добавление `_headers`, полная переработка CSS-блока во всех HTML-файлах для исправления мобильной адаптивности и обновления дизайна.
